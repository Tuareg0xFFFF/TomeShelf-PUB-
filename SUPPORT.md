# TomeShelf Support

TomeShelf is a native Audiobookshelf client for iOS, iPadOS, and macOS.
This page explains how to get help, report problems, and request features.

Last updated: May 17, 2026

## Before You Contact Us

Most TomeShelf issues are connection or server-side problems. Please check:

1. Your Audiobookshelf server is reachable from your device (try the
   server URL in Safari on the same network).
2. Your username, password, and server URL are correct.
3. You are running a supported Audiobookshelf server version
   (see https://www.audiobookshelf.org).
4. Your device has a working internet or local-network connection.

## Reporting a Bug

When opening an issue, please include:

- TomeShelf version (Settings → About)
- iOS / iPadOS / macOS version
- Device model
- Audiobookshelf server version
- Steps to reproduce the problem
- What you expected to happen
- What actually happened
- Screenshots or a screen recording, if relevant

Do **not** include your server URL, username, password, or authentication
token in public issues.

## Frequently Asked Questions

**Does TomeShelf work without an Audiobookshelf server?**
No. TomeShelf is a client for Audiobookshelf and requires a running
Audiobookshelf server. Learn more at https://www.audiobookshelf.org.

**Can I use TomeShelf with multiple servers?**
Yes. You can add multiple servers and switch between them in the app.

**How do I delete my data?**
Remove the server from the app, disable offline library to remove
downloaded files, or delete the app to remove everything including
Keychain entries. See PRIVACY.md for details.

**Where do I find my Audiobookshelf server URL?**
The URL is configured by whoever set up your Audiobookshelf server.
If you are not the server admin, please ask the person who set it up.
